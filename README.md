# Info-Extraction
Implementing KMP, BM, and Regex to extract information from text. Made in Python for ITB IF2211 Course

## Prerequisites
Things below are required to be installed in your computer before you proceed and run this program

* Python 3
* flask library
* nltk library (+ punkt modules)

**This guide were made and tried on Windows Computer**

**You might need to adapt it onto your own computer if you aren't using Windows**

### Installing flask library
**Make sure you have already installed python 3 on your local computer**

Open your terminal and type these in

`pip install flask`

### Installing nltk library (+ punkt modules)
**Make sure you have already installed python 3 on your local computer**

Open your terminal and type these in

`pip install nltk`

After that, open python3 on your local computer then follow these instructions

`import nltk`

`nltk.download()`

A new windows should be opened by now, after that follow these instructions
1. Go to module tab
2. Find and choose `punkt` module
3. Press download button

Now you're ready to run the program!

## Running the Program
1. Go to the src directory
2. Open terminal there
3. Type in `python App.py`

The web page app should be opened by now
